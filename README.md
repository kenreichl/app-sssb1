# app-sssb1

Sunny Side Songs — "Back to the Garden" children's book app for mobile and tablet.

This Flutter application delivers an interactive reading experience for the first book in the Sunny Side Songs series by Ngonda Badila (Lady Moon). The app targets iOS and Android phones and tablets and is designed to let young readers, families, and communities sing along to uplifting nursery-rhyme songs that celebrate nutrition, wellness, gardening, and community.

## Features (Planned)

- Page-turning illustrated story reader optimized for phones and tablets
- Integrated audio playback for songs and narration
- Responsive layouts that adapt between mobile and tablet form factors
- Child-friendly navigation and interaction patterns

## Project Structure

- `lib/` — Dart application code
- `assets/images/` — Illustrations and visual assets
- `assets/audio/` — Soundtracks and narration
- `assets/data/` — Structured content (lyrics, page data, etc.)
- `android/`, `ios/` — Native platform projects
